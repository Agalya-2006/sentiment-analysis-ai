# Sentiment Analysis Tool

This project is a simple Sentiment Analysis tool built using Python.

It analyzes the sentiment of a sentence and classifies it as:
- Positive
- Negative
- Neutral

## Technologies Used
- Python
- TextBlob
- Natural Language Processing (NLP)

## How to Run

1. Install the required library:
pip install textblob

2. Run the program:
python app.py

3. Enter a sentence and see the sentiment result.

## Example

Input:
I love artificial intelligence

Output:
Positive Sentiment
